## Software de gestión de datos

El proyecto **SOGEDA** nació a partir de la idea de gestionar datos personales de la información de clientes guardados en archivos csv. 
La primera versión se creó el *27 de enero de 2024* utilizando el framework Laravel, pero debido a la complejidad para usarlo en el frontend, el *1ro de marzo de 2024* se decidió por utilizar el framework React.js usando el administrador **CoreUI** para el front debido a la urgencia del producto, y como servidor de base de datos se utilizó Node.js.

La versión beta se la comenzó a utilizar el miércoles 13 de marzo de 2024 iniciando su fase de testeo mostrando varios errores en la lógica de programación, pero con resultados satisfactorios para su comienzo. Finalmente, la versión beta se completó el domingo *17 de marzo de 2024* prometiendo mejorar la calidad de los datos gestionados tanto por el cliente como por el software y se espera utilizar el framework Laravel para la parte del backend.

Agradezco a todas las personas que me apoyaron en este proyecto y espero en el futuro mejorar la calidad del software en versiones posteriores. Muchas gracias.

Link del admin de React CoreUi: https://coreui.io/react/docs/getting-started/introduction/
